Triangle.js

A smart and tiny, no-dependency JavaScript/CSS library for creating beautiful colored triangle backgrounds.

Attention! Even if Triangle.js does a great job at the moment, it is still in beta. Please report any bugs or problems on GitHub.

Installation

The installation of Triangle.js is quite easy:

  • The manual way: Download Triangle.js from the github repository and save it to your project structure.
  • Installation with bower: bower install trianglejs
  • Installation with npm: npm install trianglejs

After installing Triangle.js, easily include the required scripts to your website. Copy the stylesheet link into the <head>-area before your custom CSS:

<link rel="stylesheet" href="PATH/TO/triangle.css" />

Triangle.js's JavaScript was built without any dependencies. The only thing you have to do is including the following <script> tag near the end of your page, right before the closing </body> tag:

<script src="PATH/TO/triangle.js"></script>

Initialization

Using Triangle.js is easy. First, initialize the JavaScript by pasting the following lines below the JavaScript <script> tag included above:

var triangle = new Triangle();

Options

Name Type Default Description
colors [String] ['#ffffff', '#ff8a64', '#ffca27', '#f57b01', '#039be5'] The color palette for Triangle.js.
marginTop String "30vw" Margin-top value for triangle background.
triangleHeight String "60vw" Height value for each background tile.
preloadCount int 2 Amount of background tiles to preload. Useful in combination with dynamic page content like accordions.
animateOnChange boolean true Enables or disables CSS transition on color palette change (call of changeStyle()).
debug boolean false Enables output of some debug messages.

Methods

Name Type Description
recalculate no parameter Method to recalculate the triangle background with given parameters. Very useful when manipulating the DOM via AJAX oder animating various site elements which differ in height and the preload option is not useful.
changeStyle [string] Method to dynamically change the triangle colors. Example can be found on this site, where colors are changing every 10 seconds.
